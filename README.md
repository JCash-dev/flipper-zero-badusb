# Flipper-Zero-BadUSB
Proyecto académico para grado de las Unidades Tecnologicas de Santander, enfocado en ciberseguridad y demostraciones ética de ataques BadUSB con Flipper Zero. Solo para fines educativos y con autorización.


#  Política de Uso Ético

 Propósitos Aprobados
- Investigación académica autorizada
- Pruebas de penetración con consentimiento
- Educación en ciberseguridad
- Ejercicios de Red Team en entornos controlados

Usos Prohibidos
- Actividades maliciosas o criminales
- Pruebas sin autorización explícita
- Violación de leyes locales
- Distribución sin contexto ético
